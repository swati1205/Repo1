1. kubectl is a command line interface for running commands against Kubernetes clusters. 
2. kubectl run first-deployment --image=katacoda/docker-http-server --port=80 -: Run a specified image on the cluster.
3. kubectl get pods -: List one or more resources.
4. kubectl expose deployment first-deployment --port=80 --type=NodePort -: Expose a replication controller, service, or pod as a new Kubernetes service.
